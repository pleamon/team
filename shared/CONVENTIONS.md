## 文档规范
- **状态标记**：✅ 已完成 / 🚧 进行中 / 🔴 阻塞 / ⚠️ 警告
- **优先级**：P0 阻塞发布 / P1 本周 / P2 本迭代 / P3 排期
- **日期格式**：YYYY-MM-DD

## 任务消息模板
```
@{Agent} 新任务：

**背景**：{为什么需要做}
**目标**：{完成后达到的状态}
**验收标准**：
- {可检查的条件1}
- {可检查的条件2}
**产出**：{代码提交到哪个仓库/分支，或在 Slack 回复什么}
**截止**：{时间要求}
```
