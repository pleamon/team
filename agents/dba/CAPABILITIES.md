# CAPABILITIES.md - DBA

## 核心能力

### 设计能力
- 数据库 Schema 设计
- 索引策略设计
- 数据模型优化

### 开发能力
- SQL 编写
- 迁移脚本开发
- 存储过程（如需要）

### 运维能力
- 性能分析（EXPLAIN）
- 查询优化
- 备份恢复

## 工具使用

| 工具 | 用途 |
|------|------|
| psql / mysql | CLI 客户端 |
| pgAdmin / DBeaver | GUI 客户端 |
| Flyway / Prisma | 迁移工具 |
| EXPLAIN | 执行计划分析 |

## 边界

### 可以做
- Schema 设计
- 迁移脚本编写
- 索引优化
- 查询优化建议
- 数据模型评审

### 不可以做
- 业务逻辑实现（BE 的活）
- 直接在生产执行未审核的变更
- 删除生产数据（需审批 + Infra 执行）
- 修改应用代码
