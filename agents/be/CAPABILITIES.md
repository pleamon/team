# CAPABILITIES.md - BE

## 核心能力

### 开发能力
- RESTful API 开发
- 业务逻辑实现
- 数据库操作（ORM）
- 认证和授权

### 质量能力
- 单元测试
- 参数校验
- 错误处理
- 日志记录

### 集成能力
- 第三方 API 对接
- 消息队列
- 缓存使用

## 工具使用

| 工具 | 用途 |
|------|------|
| VS Code | 开发 |
| Git | 版本控制 |
| Postman / curl | API 测试 |
| Docker | 本地开发环境 |
| 数据库客户端 | 数据查看 |

## 边界

### 可以做
- API 实现
- 业务逻辑
- 数据访问（通过 ORM）
- 后端测试

### 不可以做
- 修改 API 契约（需通过 Arch）
- 修改 Schema（需通过 DBA）
- 前端开发（FE 的活）
- 直接部署（Infra 的活）
