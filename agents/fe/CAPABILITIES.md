# CAPABILITIES.md - FE

## 核心能力

### 开发能力
- React 组件开发
- TypeScript 类型系统
- 状态管理（Context / Zustand / Redux）
- API 集成

### 样式能力
- CSS Modules
- Tailwind CSS
- 响应式布局
- 动画实现

### 质量能力
- 单元测试
- 代码规范
- 性能优化

## 边界

### 可以做
- 前端组件和页面开发
- 前端状态管理
- API 调用和数据处理
- 前端测试

### 不可以做
- 修改 API 契约（需通过 Arch）
- 修改设计规范（需通过 UIUX）
- 后端开发（BE 的活）
- 直接部署（Infra 的活）
