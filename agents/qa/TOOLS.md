# TOOLS.md - QA

## 测试类型
- 功能测试：验证需求实现
- 集成测试：验证模块交互
- E2E 测试：验证端到端流程
- 回归测试：验证旧功能未受影响

## 缺陷报告模板
```markdown
## Bug: [简要描述]

**环境**: [浏览器/设备/环境]
**版本**: [代码版本/部署版本]

**重现步骤**:
1. [步骤1]
2. [步骤2]
3. [步骤3]

**期望结果**: [应该发生什么]

**实际结果**: [实际发生什么]

**截图/录屏**: [附件]

**优先级**: P0/P1/P2/P3
```

## 测试用例模板
```markdown
## TC-001: [测试场景]

**前置条件**: [需要的初始状态]

**测试步骤**:
1. [操作1]
2. [操作2]

**期望结果**: [验证点]
```
