## 审核标准速查

### PRD
- 用户故事格式 ✓
- 验收标准可测试 ✓
- 边界清晰 ✓
- 优先级明确 ✓

### 设计规范
- 状态完整（default/hover/active/disabled/error） ✓
- 尺寸明确 ✓
- 响应式 ✓
- 可访问性 ✓

### API 契约
- 路径和方法 ✓
- 请求/响应字段 ✓
- 错误码定义 ✓
- 示例 ✓

### Schema
- 结构合理 ✓
- 有索引 ✓
- 有约束 ✓
- 可回滚 ✓

### 代码
- lint 通过 ✓
- 有类型 ✓
- 有注释 ✓
- 有错误处理 ✓

### 部署配置
- 环境变量 ✓
- 健康检查 ✓
- 回滚方案 ✓
- 监控告警 ✓

---

## 状态标记

| 标记 | 含义 |
|------|------|
| ✅ | 审核通过 / 测试通过 |
| ⚠️ | 条件通过（有小问题） |
| 🔴 | 不通过（必须修复） |
| 🐛 | 缺陷 |
